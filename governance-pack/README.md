# Governance Pack

This folder contains template-owned assets that do not need to live at repository root.

## Contents

- `templates/`: optional starter templates (lint presets, etc.)
- `scripts/`: script entrypoints for checks, validation, and bootstrap flow
- `docs/`: template-owned documentation and changelog

Root-path-dependent files remain at root (for tool compatibility), such as:

- `.github/` workflows and templates
- `.template/` configuration files
- `lefthook.yml`
- `.editorconfig`

# Template Changelog

Use this file as a reference template for writing changelog entries.
The actual template release history is maintained in `CHANGELOG.md` at the repository root (mirrored under `docs/operations/tooling-changelog.md`).

This format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
and [Semantic Versioning](https://semver.org/spec/v2.0.0.html).
